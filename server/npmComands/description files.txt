dote.env - настроки к подключению к БД, описание переменных окружения
models.js - описание модели данных с таблиц
cors - возможность отправлять запросы с браузера
routes - навигация по сайту
routes => controllers разделение функциональности, чтобы не мусорить все в один файл, функции служат как параметры в методы в роутеров,
при открытии определенных урд эти функции отрабатывают
